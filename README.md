# تطبيق القرآن الكريم - الشيخ محمود خليل الحصري (PWA)

تطبيق ويب تقدمي (PWA) للاستماع إلى القرآن الكريم بتلاوة الشيخ محمود خليل الحصري، مصمم ليعمل بكفاءة عالية وبدون إنترنت.

## المميزات

- **استماع بدون إنترنت**: يتم حفظ المقاطع الصوتية تلقائياً عند تشغيلها لأول مرة لتتمكن من سماعها لاحقاً بدون اتصال.
- **إدارة التحميلات**: يمكنك تحميل الحزب كاملاً أو جميع أحزاب القرآن الستين (480 ملف صوتي) بضغطة واحدة.
- **تصميم إسلامي فاخر**: واجهة مستخدم داكنة مريحة للعين مع لمسات ذهبية وجماليات إسلامية.
- **سهولة التنقل**: تنظيم دقيق حسب الأحزاب والأثمان (8 أثمان لكل حزب).
- **مشغل صوت متطور**: يدعم التشغيل التلقائي للمقطع التالي والتحكم في التقديم والتأخير.
- **تطبيق ويب تقدمي (PWA)**: يمكن تثبيت التطبيق على هاتفك أو حاسوبك كأنه تطبيق أصلي.

## المتطلبات التقنية

- **Next.js 16** (App Router)
- **Serwist** (لإدارة التخزين المؤقت والعمل بدون إنترنت)
- **CSS** (للتصميم)

## كيفية الاستخدام

1. افتح التطبيق في المتصفح.
2. اختر الحزب المراد سماعه.
3. اضغط على الثمن المطلوب لبدء التشغيل.
4. للتحميل المسبق، اضغط على زر "تحميل" في الأعلى واختر "تحميل الكل" أو اختر أحزاباً معينة.

## تطوير التطبيق

إذا كنت مطوراً وترغب في تشغيل المشروع محلياً:

```bash
# تثبيت الاعتمادات
npm install

# تشغيل خادم التطوير
npm run dev
```

سيظهر التطبيق على الرابط: `http://localhost:3000`
